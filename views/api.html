Submitting A Report

 Submit data which is required to report an incident

Request URL

POST: http://54.186.110.31/submitReport

Request parameters

Parameter Value Description Required
person ENUM:
SURVIVOR, FRIEND_OF_SURVIVOR, RELATIVE_OF_SURVIVOR, NOT_RELATED
This field takes one of the above mentioned values		 This is to get more information about the person YES
doYouKnow ENUM:
Y, N
This field takes one of the above mentioned values		 This is to get more information about the person YES
firstTimeCrime ENUM:
Y, N, X
This field takes one of the above mentioned values		 Setting the field to X means that the question is not applicable NO
incidentList String: Comma seperated string of the following ENUMS
EVE_TEASING, VOYEURISM, ACID_VIOLENCE, STALKING, RAPE, DISROBING, DOMESTIC_VIOLENCE, MARITAL_RAPE, OTHER_INCIDENT
Eg: EVE_TEASING, VOYEURISM		 It indicates the type of incident that happended with the victim YES
otherIncidence String Description about the other incident that happend. This field is only set when OTHER_INCIDENT is passed in incidentList field NO
location String Location of the incident YES
locationLat String Latitude of the location YES
locationLng String Longitude of the location YES
incidentDate String: Format YYYY-MM-DD
Eg: 2013-02-13 		Date of the incident YES
incidentTime String: Format HH:MM
Eg: 12:23 		Time of the incident YES
comments String Provide additional miscellenous comments about the incident NO
firstName String First Name of the reportee NO
lastName String Last Name of the reportee NO
email String Email of the reportee NO
number String Phone number of the reportee NO

Sample Usage

Using cURL: curl -X POST http://54.186.110.31/submitReport --header "Content-Type:application/json" -d @sampleDataRequest.json

Get Reported Incidents

 Returns the Latitude, Longitude and Category of all reported incidents

Request URL

GET: http://54.186.110.31/viewReport

Request parameters

Not Applicable

Response fields

Field Description
latitude Latitude of the location
longitude Longitude of the location
categories category of the incident

Sample Usage

Using cURL: curl http://54.186.110.31/viewReport

Sample Response

{ "latitude": "19.1171389", "longitude": "72.846563", "categories": "ACID_VIOLENCE" },
{ "latitude": "19.05", "longitude": "72.8333", "categories": "EVE_TEASING,STALKING,RAPE" }
