Submitting A Report

Submit data which is required to report an incident

Request URL

POST: http://54.186.110.31/submitReport

Request parameters

Parameter Value Description Required
person ENUM:
SURVIVOR, FRIEND_OF_SURVIVOR, RELATIVE_OF_SURVIVOR, NOT_RELATED
This field takes one of the above mentioned values		 This is to get more information about the person YES
doYouKnow ENUM:
Y, N
This field takes one of the above mentioned values		 This is to get more information about the person YES
firstTimeCrime ENUM:
Y, N
This field takes one of the above mentioned values		 To indicate whether is a first time crime. If this field is left blank, we consider the question as Not Applicable NO
incidentList String: Comma seperated string of the following ENUMS
EVE_TEASING, VOYEURISM, ACID_VIOLENCE, STALKING, RAPE, DISROBING, DOMESTIC_VIOLENCE, MARITAL_RAPE, OTHER_INCIDENT
Eg: EVE_TEASING, VOYEURISM		 It indicates the type of incident that happended with the victim YES
otherIncidence String Description about the other incident that happend. This field is only set when OTHER_INCIDENT is passed in incidentList field NO
location String Location of the incident YES
locationLat float Latitude of the location NO
locationLng float Longitude of the location NO
incidentDate String: Format YYYY-MM-DD
Eg: 2013-02-13 		Date of the incident NO
incidentTime String: Format HH:MM:SS
Eg: 12:23:11 		Time of the incident NO
comments String Provide additional miscellenous comments about the incident NO
firstName String First Name of the reportee NO
lastName String Last Name of the reportee NO
email String Email of the reportee NO
number String Phone number of the reportee NO
